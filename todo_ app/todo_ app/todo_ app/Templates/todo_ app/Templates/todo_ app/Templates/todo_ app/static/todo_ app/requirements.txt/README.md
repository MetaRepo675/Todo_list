# Task Management System (Todo List)

A simple and functional task management application with Flask and SQLite.

## Features

- ✅ Add new task with priority and due date
- ✅ Edit existing tasks
- ✅ Mark tasks as completed
- ✅ Filter tasks (all, pending, completed)
- ✅ Change task priority
- ✅ Persian and responsive user interface
- ✅ Delete tasks with confirmation
